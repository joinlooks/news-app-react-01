# React News App

Visit here: [react-news-app](https://joinlooks.github.io/news-app-react-01/)

## Usage

- To run on your local machine

```
$ git clone https://github.com/joinlooks/news-app-react-01.git
$ yarn
$ yarn dev
```

- Then visit the site printed in terminal.
